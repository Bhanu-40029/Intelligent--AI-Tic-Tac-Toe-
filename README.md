# AI-Powered Tic-Tac-Toe Visualizer

An interactive AI-based Tic-Tac-Toe Visualizer developed using **Python (Flask)** and **HTML, CSS, and JavaScript**. The project demonstrates various Artificial Intelligence algorithms through real-time game tree visualization, search traversal, probabilistic reasoning, and explainable decision-making.

##  Features
- Interactive Tic-Tac-Toe Gameplay
- BFS, DFS, UCS, Greedy BFS, and A* Search
- CSP (Backtracking)
- Minimax Algorithm
- Conditional Probability
- Bayes Theorem
- Bayesian Network
- Game Tree Visualization
- Explainable AI Decision Support

## 🛠️ Technologies Used
- Python
- Flask
- HTML5
- CSS3
- JavaScript
- PyCharm
